# manufacturing-intelligence-platform
This project develops an end-to-end Manufacturing Intelligence Platform that integrates production, quality, inventory, maintenance, and supply chain data into a centralized analytics system.
The platform includes:
- Data Engineering (ETL)
- PostgreSQL Data Warehouse
- Machine Learning
- Predictive Maintenance
- Production Analytics
- Quality Analytics
- Inventory Analytics
- Power BI Dashboards
- FastAPI Backend
- Streamlit Application
- AI Assistant using LangGraph
## Tech Stack
- Python
- Pandas
- PostgreSQL
- Scikit-learn
- FastAPI
- Power BI
- Streamlit
- Docker
- GitHub
